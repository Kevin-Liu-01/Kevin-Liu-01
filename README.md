## <img src="https://raw.githubusercontent.com/Tarikul-Islam-Anik/Animated-Fluent-Emojis/master/Emojis/Objects/Hammer%20and%20Wrench.png" width="22"/> what I'm working on right now

- **[Dedalus Labs](https://dedaluslabs.ai)** — model-agnostic, MCP-powered agent SDK. Built the multi-tenant auth layer (DAuth) and the sandboxed execution surface on microVM workspaces.
- **[CoTCodec](https://github.com/Kevin-Liu-01)** — research with Prof. Danqi Chen treating language choice as an orchestration variable for tool-using agents. Builds on ReAct, AutoGen, LLMCompiler.
- **[Princeton Tower Defense](https://princetontd.vercel.app)** — a Kingdom Rush clone for Princetonians. Because someone had to.
- **Personal LLM Wiki** — 229 pages, 99 skills, 14 cron-driven automations, local-first hybrid search via [qmd](https://github.com/tobi/qmd). The brain compounds.
